# UI 元件庫

此目錄包含由 shadcn/ui 提供的基礎 UI 元件，構成應用程式的設計系統基礎。

## 元件列表

- **`accordion.tsx`** - 手風琴元件
- **`alert.tsx`** - 警告提示元件
- **`alert-dialog.tsx`** - 警告對話框
- **`avatar.tsx`** - 頭像元件
- **`badge.tsx`** - 徽章元件
- **`button.tsx`** - 按鈕元件
- **`calendar.tsx`** - 日曆元件
- **`card.tsx`** - 卡片元件
- **`carousel.tsx`** - 輪播元件
- **`chart.tsx`** - 圖表元件
- **`checkbox.tsx`** - 複選框元件
- **`collapsible.tsx`** - 可折疊元件
- **`context-menu.tsx`** - 右鍵選單
- **`dialog.tsx`** - 對話框元件
- **`dropdown-menu.tsx`** - 下拉選單
- **`form.tsx`** - 表單元件
- **`input.tsx`** - 輸入框元件
- **`label.tsx`** - 標籤元件
- **`menubar.tsx`** - 選單欄
- **`popover.tsx`** - 彈出框
- **`progress.tsx`** - 進度條
- **`radio-group.tsx`** - 單選框組
- **`scroll-area.tsx`** - 滾動區域
- **`select.tsx`** - 選擇器
- **`separator.tsx`** - 分隔線
- **`sheet.tsx`** - 側邊面板
- **`sidebar.tsx`** - 側邊欄
- **`skeleton.tsx`** - 骨架屏
- **`slider.tsx`** - 滑塊
- **`switch.tsx`** - 開關
- **`table.tsx`** - 表格
- **`tabs.tsx`** - 標籤頁
- **`textarea.tsx`** - 多行文本輸入
- **`toast.tsx`** - 提示訊息
- **`toaster.tsx`** - 提示容器
- **`tooltip.tsx`** - 工具提示

## 用途

- 提供一致的設計語言
- 支援快速 UI 開發
- 確保元件可訪問性
- 維護設計系統標準
