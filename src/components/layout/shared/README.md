# Shared 共享佈局元件

此目錄包含可在應用程式各處重用的通用佈局元件。

## 元件

- **`empty-state.tsx`** - 空狀態元件
- **`logo.tsx`** - Logo 元件
- **`page-container.tsx`** - 頁面容器
- **`page-header.tsx`** - 頁面標題
- **`section-divider.tsx`** - 區段分隔線
- **`status-indicator.tsx`** - 狀態指示器

## 用途

- 提供一致的頁面佈局
- 支援頁面標題和容器
- 實現狀態顯示和指示
- 改善用戶界面的視覺一致性
