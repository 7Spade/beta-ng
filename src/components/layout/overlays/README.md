# Overlay Components 目錄

此目錄包含應用程式的覆蓋層元件，用於顯示模態對話框、彈出選單等。

## 主要元件

- **`drawer-container.tsx`**: 抽屜容器元件
- **`modal-container.tsx`**: 模態對話框容器
- **`popover-container.tsx`**: 彈出選單容器
- **`tooltip-provider.tsx`**: 工具提示提供者

## 使用方式

這些元件用於創建層級化的使用者介面，提供豐富的互動體驗。

## 設計原則

- 一致的覆蓋層行為
- 支援鍵盤導航
- 適當的焦點管理
- 響應式設計
- 無障礙功能支援
